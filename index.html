GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens，权限勾选 Contents (Read and Write)

📋 使用指南：
1. 首次使用：填写 Token 和仓库信息 → 点击"保存并连接" → 数据自动同步
2. 换电脑登录：在新浏览器打开系统 → 进入后台 → 填写相同 Token → 点击"从云端拉取"
3. 数据冲突：若本地与云端不一致，以云端为准，可手动"强制推送"覆盖
4. 安全建议：仓库设为私有，Token 定期轮换
